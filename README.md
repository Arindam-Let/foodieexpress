# FoodieExpress – Online Food Ordering System

FoodieExpress is a simple online food ordering website where users can explore menus, create accounts, log in, and place food orders. This project was developed as part of a BTech web development project.

## Technologies Used
Frontend:
- HTML
- CSS
- Bootstrap
- JavaScript
Backend:
- PHP

Database:
- MySQL

Server:
- XAMPP

## Features

- User Signup
- User Login
- Browse Food Menu
- Place Food Orders
- Contact Form
- Responsive Design
- Video and Image Content


## How to Run the Project

1. Install XAMPP.
2. Copy the project folder into: xampp/htdocs/
3. Start Apache and MySQL in XAMPP.
4. Import the database in phpMyAdmin.
5. Open the browser and go to: http://localhost/foodieexpress
